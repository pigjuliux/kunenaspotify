KUNENASPOTIFY - QUICK INSTALLATION GUIDE



This guide is meant to be a quick reference for site administrators.

However, the process doesn't differ from the installation of a standard Joomla Plugin.

Check the file "REQUIREMENTS.txt" in order to learn more about the version of Joomla and Kunena which have been tested.

Also, be sure to read and understand the file "LICENSE.txt", especially the part regarding the use of the Spotify Widget inside your site.



Open the Joomla Administration Page and reach the "Extensions -> Plugins" settings.

On the "Install" tab, select (from your disk) the ".zip" file of the KunenaSpotify plugin.

Click the "Upload & Install" button and ensure that Joomla doesn't report any error.

Go to "Kunena Forum"->"Plugins".

As you can see from the attached image, the plugin is there but is currently disabled.

Click the "red crossed" button in order to enable it

If everything is ok, users should see the new BBcode icon while posting a new message.

The code typed inside "[spotify][/spotify]", is the trackcode retrieved from Spotify itself.
